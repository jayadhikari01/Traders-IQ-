# 🏛️ Traders IQ | The Elite Terminal

**Traders IQ** is a high-performance trading dashboard and psychological journaling ecosystem designed for professional traders. Engineered for precision and discipline, it combines sleek UI/UX with robust financial analytics to eliminate guesswork from the markets.

---

## ⚡ Key Features

- **🎯 Smart PnL Analytics:** Real-time data visualization of your win rate, profit targets, and daily performance.
- **🧠 Market Psychology Gauge:** Advanced tools to monitor emotional discipline and maintain a logical trading mindset.
- **☁️ Cloud Safe Sync:** Secured by Firebase, ensuring your journal and trade data are synced globally across devices.
- **🏆 Global Leaderboard:** Unlock 100+ achievement badges and compete with the elite trading community.
- **💎 Modern Aesthetic UI:** A 2026-standard "App-vibe" interface built with Tailwind CSS and Plus Jakarta Sans typography.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS (Modern Mesh Gradients)
- **Backend:** Firebase (Authentication & Firestore)
- **Payment Gateway:** Lemon Squeezy (Merchant of Record)
